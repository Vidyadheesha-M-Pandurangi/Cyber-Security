This project implements an AI-powered Network Intrusion Detection System (NIDS) using Machine Learning (Random Forest) and deploys it as an interactive Streamlit web application.
The system is designed to detect malicious network traffic, highlight the most influential features used for detection, and provide real-time analysis through a clean, user-friendly interface.

The project is suitable for cybersecurity demonstrations, academic projects, and internship evaluations.

The project uses a publicly available network traffic dataset containing benign and malicious flows. Due to its large size, the dataset is hosted externally on Hugging Face and loaded dynamically at runtime.

Dataset link: "https://huggingface.co/datasets/vidyadheesha-m-pandurangi/Cybersecurity-Datasets/resolve/main/Dataset.csv"
